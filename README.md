# Product-Recommender-System
This is a product recommender system using django
Software implementation 

Installation Links:
1.	How to install Django
https://docs.djangoproject.com/en/2.0/topics/install/
2.	Install Pillow: Image library for django
http://pillow.readthedocs.io/en/3.4.x/installation.html 
3.	Install Celery: 
Celery is a task queue with batteries included. It’s easy to use so that you can get started without learning the full complexities of the problem it solves. It’s designed around best practices so that your product can scale and integrate with other languages, and it comes with the tools and support you need to run such a system in production.
http://docs.celeryproject.org/en/latest/getting-started/first-steps-with-celery.html   
4.	Install Flower: Flower is a web based tool for monitoring and administrating Celery clusters
http://flower.readthedocs.io/en/latest/install.html 

5.	Downloading and Installing RabbitMQ: RabbitMQ is an open source message broker software (sometimes called message-oriented middleware) that implements the Advanced Message Queuing Protocol (AMQP). 
https://www.rabbitmq.com/download.html 

6.	Install Django PayPal: A pluggable application that implements with PayPal Payments Standard and Payments Pro. 
https://django-paypal.readthedocs.io/en/stable/

7.	Install django-parler: Django Parler provides Django model translations 
https://django-parler.readthedocs.io/en/latest/ 

8.	Install Redis : Redis is an open source (BSD licensed), in-memory data structure store, used as a database, cache and message broker. It supports data structures such as strings, hashes, lists, sets, sorted sets with range queries, bitmaps, hyperloglogs and geospatial indexes with radius queries.
https://redis.io/download
